# GUI
GUI project-Smart Bus Reservation System

This Smart Bus Reservation System was developed to book a bus seat through a web or desktop application and manage schedules using HTML, CSS, JavaScript, and c#. Web Features: Such features comprise the ability to book a seat during any instant, register a user, view a route, and create a route. Desktop Features: A management interface or admin dashboard that enables managing schedules for buses, routes, available seats, and comments left by consumers.
